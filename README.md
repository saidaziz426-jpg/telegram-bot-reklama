# Reklama O'chiruvchi Bot

Bu bot guruhlardagi reklama, linklar va spam xabarlarni avtomatik ravishda o'chiradi.

## O'rnatish

1.  Python o'rnatilganligiga ishonch hosil qiling.
2.  Kerakli kutubxonalarni o'rnating:
    ```bash
    pip install -r requirements.txt
    ```

## Ishga tushirish

Botni ishga tushirish uchun quyidagi buyruqni bering:

```bash
python main.py
```

## Botni guruhga sozlash

1.  Botni guruhingizga qo'shing.
2.  Botni guruhda **Admin** qilib tayinlang (Xabarlarni o'chirish huquqi bilan).
3.  Bot endi ishlashga tayyor!

## 24/7 Ishlatish

Bot doimiy ishlashi uchun uni biror serverga (VPS) joylashtirish kerak. Masalan:
-   **Railway**
-   **Heroku**
-   **PythonAnywhere**
-   Yoki o'zingizning doimiy yoniq turadigan kompyuteringizda.

## Xususiyatlari

-   Http/Https linklarni o'chiradi.
-   Telegram linklarini (t.me) o'chiradi.
-   Mention (@username) reklamalarni o'chiradi.
-   Reklama kalit so'zlari (sotiladi, kanalga obuna va h.k.) asosida filtrlaydi.
